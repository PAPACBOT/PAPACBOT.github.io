PAPAC Features

Google Stuff

  • $search QUERY - Search on Google
  • $ytsearch QUERY - Search on Youtube
  • $isearch QUERY - Search on Google Images

Role Commands

  • $addrole ROLE_ID - Add Role to replied message author
  • $removerole ROLE_ID - Removes Role from replied message author

Nickname Commands

  • $setnick NICKNAME - Changes Bot's nickname
  • $nickuser NICKNAME - Changes Replied message author Nickname

Information Commands

  • $status SERVER_ADDRESS - Gets a Minecraft Server status from IP Address
  • $getinfo USER_ID - Gets User information from ID
  • $invites - Gets Replied Message Author's invites

Comment Commands

  • $vouch Comment Attachment - Submits a Comment!
  • $setvouch - Sets Current Channel as Vouch Log Channel!

Channel Management Commands

  • $lock - Makes Users Unable to send message
  • $unlock - Makes Users be able to send message
  • $hide - Makes Users Unable to view channel
  • $unhide - Makes Users be able to view channel
  • $newticket - Makes New Ticket
  • $renamec NEW_NAME - Renames Channel
  • $removec - Deletes Current channel

Crypto Help

  • $baltc LTC_ADDRESS - Gets Balance of Litecoin Address
  • $babtc BTC_ADDRESS - Gets Balance of Bitcoin Address
  • $calc ltc-to-$/$-to-ltc - Converts LTC/$ to $/LTC
  • $calc btc-to-$/$-to-btc - Converts BTC/$ to $/BTC
